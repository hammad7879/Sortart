# Sortart
1. Executive Summary
Sortart is an advanced digital asset management ecosystem designed specifically for the modern creator. It moves beyond traditional "file storage" by integrating generative AI, real-time color theory tools, and professional portfolio hosting into a single, high-performance interface. Developed using a cutting-edge "AI-first" workflow with v0 and Cursor, Sortart eliminates the administrative friction of the creative process, allowing artists to spend more time in their flow state.

2. Core Problem & Mission
Traditional organizational tools are built for documents and spreadsheets, failing to meet the visual and intuitive needs of artists.

"We are making life easy for the people who color our world—the artists and creators who shouldn't have to fight with messy folders just to find their own inspiration."

3. Key Feature Suite
AI Creative Co-Pilot: An integrated chatbot powered by OpenAI and Claude that assists with brainstorming, project naming, and workflow automation.

Neural Search (Exa Integration): Unlike standard keyword search, our integration with Exa allows users to find assets and external inspiration using semantic meaning and context.

Professional Color Lab: * Color Editor: A precision tool for manipulating hex codes, RGB values, and HSL levels.

Color Mixer: An interactive canvas to blend colors and generate unique, harmonious palettes.

User Ecosystem: Personalized User Profiles and a Saved Portfolio feature that allows creators to transition instantly from "working mode" to "showcase mode."

4. Technical Architecture
The project is built on a robust, scalable foundation designed for speed and developer efficiency:

Framework: Next.js ensures a lightning-fast, SEO-friendly, and responsive user experience.

Monorepo Structure: The project is organized as a monorepo (root package.json) with the core application logic isolated in the Sortart/ directory.

Dependency Management: Centralized scripts handle everything from environment preparation to production builds (npm run build --prefix Sortart), ensuring a clean and reliable deployment pipeline.

Modern Tooling: Developed utilizing Cursor for AI-assisted coding and v0 for rapid, high-fidelity UI component generation.

5. Development Workflow
The repository is configured for professional-grade development:

npm run dev: Launches the local development environment within the application subdirectory.

npm run lint: Ensures code quality and consistency across the entire codebase.

npm run prepare: Automates dependency installation for the nested application, lowering the barrier for new contributors.

6. Future Vision
Sortart aims to become the industry standard for "living" portfolios—where the tools used to create the art live side-by-side with the art itself. We plan to expand our AI capabilities to include automated asset tagging and style-based grouping, further cementing our commitment to an effortless creative experience.
